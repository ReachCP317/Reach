1. Introduction

Reach is being developed both as an Android app and a website. The website is able to run on any modern browser, while the app is able to run on any Android version 15.0 or higher.
In both cases, the device must be able to utilize Google Maps.

1.1 Purpose

This document describes Reach's system design and architecture.

5. Detailed Design

Detailed description of important functions for Reach.

onMapReady(GoogleMap)

Called when the map is ready to be used. Overriden to call updateEventsList(), setMyLocationEnabled().

onMarkerClick(Marker)

Called when a marker is clicked/tapped. This will be overridden to go to the EventActivity.

updateEventsList()

Iterate through the list of events in the database, storing only nearby locations (small difference in lat/long) into eventsList. For each event, if distanceTo(Location) returns a value (in meters, will have to convert) less than or equal to the user's radius, a Marker will be placed via addMarker().

interested()

Called when the user presses the 'Interested' button. Increments the Event's "totalInterested" value by 1. When the button is pressed again, decrement by 1.

giveRating(float)

Called when the user presses the 'Rate' button. Calculates the new rating of the event's creator and updates it to the database.

setRadius(float)

Sets the user's radius to whatever value in kilometers was entered in the text box.

createEvent()

Called when the 'Create Event' button is pressed. Places a draggable marker on the user's location. Once the marker has been confirmed to be in the correct position, opens the EventView.

deleteEvent()

Called when the 'Delete Event' button is pressed. Clears all event information and removes the event from the database.

updateEvent()

Submits changes to the event's information to the database.

changeLocation()

Opens MainActivity and makes the event's Marker draggable. Once the marker has been confirmed to be in the correct position, opens the EventView.
